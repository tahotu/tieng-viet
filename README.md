# Tiezngvieft
Cách viết khác cho tiếng Việt (chỉ để giải trí)

Dùng thử tại: https://tahotu.github.io/tieng-viet/

## Giới thiệu

Cách viết tiếng Việt:
 - Chỉ dùng các chữ cái trên bàn phím qwerty tiêu chuẩn
 - Có thể viết liền những từ phức

Ví dụ:

> Đây là cách viết khác cho tiếng Việt

Dei lad kax viet khac co tiezngvieft

> Hôm qua tôi đi du thuyền ở Hạ Long, Quảng Ninh

Howmqua toiw di zuthuiedn ors Haflong, Quasngning



## Quy tắc

### 1. Đổi phụ âm

####  Phụ âm đầu
| Gốc | Mới | ví dụ |
|----|--|--| 
| gh | g | ghê → ge |
| ngh, ng | w | nghiêng -> wieng |
| nh | jn | nho -> jno |
| gi | j | gia  → ja |
| d | z | da  → za |
| đ | đ | đa  → da |
| ch | c | cha  → ca |
| ph | f | pha  → fa |
| c, k | k | ca  → ka |

#### Phụ âm cuối

| Gốc | Mới | ví dụ |
|----|--|--| 
| ch | x | sách  → sax |
| t | th | * _Nếu phụ âm đầu của tiếng tiếp theo trong từ phức bắt đầu bằng "h", "r"_ <br> kết hôn  → kethhown, sốt rét -> sowthraet |
| nh | ng, in | xinh  → xing, xanh  → xain |

- Các từ phức **viết liền không dấu cách**

  
- Nguyên âm khi đứng **đầu** ở tiếng thứ 2 trở đi trong từ phức phải có phụ âm câm "y"
```
Mai an  → Maiyan
```

### 3. Đổi dấu thanh

- Đổi dấu thanh bằng ký tự
- Ký tự dấu đứng sau vần, trước phụ âm cuối


| Dấu thanh | Mới | ví dụ |
| - | - | - |
| à (huyền) | d | hà → had, lừa → luard |
| á (sắc) | z | cá → kaz, chính → cizng |
| ả (hỏi) | s | tả → tas, chảnh → caisn |
| ã (ngã) | b | mã → mab, huyễn → huiebn |
| ạ (nặng) | f | cộng → kowfng, thạch → thafx |
